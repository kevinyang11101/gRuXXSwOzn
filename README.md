# 图书管理系统 python119

## 项目概述

图书管理系统是基于Python语言和MySQL数据库开发的一套应用软件。该系统旨在提供便捷的图书信息管理功能，支持电子文件操作。

## 技术栈

- 开发语言：Python
- 数据库：MySQL
- 开发工具：PyCharm（推荐）

## 功能模块

- 图书信息管理
- 用户权限管理
- 借阅与归还处理
- 数据统计分析

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 操作系统：兼容Windows、Linux和macOS
- Python版本：3.x
- 数据库版本：MySQL 5.x 或更高

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库
3. 配置数据库
4. 导入数据库脚本
5. 运行系统
